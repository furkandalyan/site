# TurkeyEarthquake

TurkeyEarthquake is a Django-based web project that collects, analyzes, and visualizes earthquake data in Turkey.

It includes ingestion utilities, storage, and analysis features that surface trends like magnitude distribution, frequency over time, and regional patterns. The site provides analysis, statistics, and forecasting views for exploring the dataset.

Features:
- Data ingestion and storage
- Trend and statistics analysis
- Analysis / statistics / forecasting views
- Visualization dashboard

## Tech Stack
- Django
- Data ingestion (API/CSV)
- Analytics layer
- Visualization layer

## Technical Architecture
`Data Sources -> Ingestion -> DB -> Analytics -> Django Dashboard`

## Challenges & Lessons Learned
- Data quality and cleaning
- Time-series performance and aggregation
- Dashboard readability for dense data

## Testing / Logging / Monitoring
- Tests: Manual checks with sample datasets
- Logging: Ingestion and error logs
- Monitoring: Hosting metrics (optional)
