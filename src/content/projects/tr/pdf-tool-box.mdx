# PDF TOOL BOX

PDF TOOL BOX, PDF dosyalarini yonetmek ve duzenlemek icin modern bir masaustu uygulamasidir.

Koyu tema arayuzu ile birlestirme, bolme, sayfa silme, toplu sifreleme/dondurme, OCR ile metin cikarma ve metadata duzenleme gibi bircok ozellik sunar.

Ozellikler:
- PDF birlestirme (ozel siralama)
- Araliga veya yer imlerine gore bolme
- Sayfa silme
- Toplu sifreleme ve dondurme
- OCR metin cikarma
- Metadata duzenleme
- Undo/Redo
