# Perfect Timer Game

Perfect Timer Game (Tkinter). Python ve Tkinter ile gelistirdigim hassas zamanlama oyunu.

Amac basit: hedef sure belirle ve kronometreyi tam aninda durdur. Ne erken ne gec.

Ozellikler:
- Hedef sure belirleme
- Milisaniye hassasiyetinde zamanlama
- Basit, odakli oyun mekaniği
