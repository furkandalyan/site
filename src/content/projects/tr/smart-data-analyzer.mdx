# Smart Data Analyzer

Smart Data Analyzer, Python ve Streamlit ile gelistirilmis modern bir veri kesif uygulamasidir.

Kod yazmadan veri analizi yapmayi kolaylastirir. CSV, Excel ve JSON dosyalarini yukleyebilir, otomatik icgoru uretir, grafikler olusturur ve PDF rapor disari aktarir.

Ozellikler:
- CSV / Excel / JSON yukleme
- Otomatik analiz ve icgoru
- Profesyonel grafikler
- PDF rapor disari aktarma
